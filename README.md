## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
pip install -r requirements.txt
```

## Running The App

```bash
./main.py
or for the server :
./main_ssl.py
```

## Viewing The App

Go to 'localhost:5000'
or '[localIP]:5000'
